# 📧 Spam Email Classifier (Machine Learning Project)

## 🌐 Live Demo
Try the app here: https://your-app.streamlit.app

## 📌 Overview
This project builds a machine learning model to classify SMS messages as **Spam** or **Not Spam** using Natural Language Processing (NLP) techniques.

## 🚀 Features
- Text preprocessing using TF-IDF vectorization
- Machine learning classification with Naive Bayes
- Model evaluation using accuracy, precision, recall, and F1-score
- Real-time prediction on custom input messages

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)

## 📊 Results
- Accuracy: ~95–98%
- Strong precision and recall for spam detection

## 🧠 How It Works
1. Clean and preprocess SMS text data  
2. Convert text into numerical features using TF-IDF  
3. Train a Naive Bayes classification model  
4. Evaluate performance on test data  
5. Predict new messages as spam or not spam  

## 💡 Example Prediction
Input:
"URGENT! You won $1000 cash! Click now!"

Output:
**Spam**
